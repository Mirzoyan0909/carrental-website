## 🚀 ДЕПЛОЙ НА БЕСПЛАТНЫЙ ХОСТИНГ:

### 1. Backend (Render.com):
```
1. Зарегистрируйтесь на Render.com
2. Создайте Web Service
3. Подключите GitHub
4. Используйте render.yaml
5. Добавьте MONGO_URL из MongoDB Atlas
```

### 2. Frontend (Netlify):
```
1. Зарегистрируйтесь на Netlify
2. Подключите GitHub
3. Используйте netlify.toml
4. Добавьте REACT_APP_BACKEND_URL (URL из Render)
```

### 3. Database (MongoDB Atlas):
```
1. Создайте бесплатный кластер M0
2. Создайте пользователя
3. Разрешите доступ со всех IP
4. Скопируйте connection string
```

Подробнее смотрите в файле **DEPLOYMENT.md**!
